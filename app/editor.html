    Treasure Factory Editor

    This editor is locked. To open it, paste a GitHub fine-grained Personal Access Token with Contents: Read and write on aphelionz/treasure-factory. The token is stored only in this browser and is verified against the repo before the editor opens; repo write access is the allowlist.

    1. GitHub → your profile photo (top-right) → Settings.
    2. Left sidebar, bottom: Developer settings.
    3. Personal access tokens → Fine-grained tokens → Generate new token.
    4. Token name: e.g. treasure-factory editor. Set an Expiration (re-issue when it lapses).
    5. Resource owner: the repo owner (aphelionz).
    6. Repository access → Only select repositories → pick treasure-factory.
    7. Repository permissions → Contents → Read and write (Metadata: Read is added automatically).
    8. Generate token, copy it, and paste it below.
